Navigation summary

  • Watch List — draggable ordered list; status badges (Ready / Downloading / Queued); "Mark Watched" on ready movies; "Download Now" override on queued ones
  • Watched — grid of completed movies; ratings revealed side-by-side once both submitted; critic quotes shown
  • Add Movie — paste an IMDB or Criterion URL; preview card loads; confirm to add
  • Browse Criterion — external link to criterion.com (opens new tab)
  • Ask Claude — external link to claude.ai with pre-filled prompt listing recent watches
